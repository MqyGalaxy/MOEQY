关于:Awiki

# Awiki 是什么？

Awiki 是一个番剧推荐 wiki 站点，偏向收集日本动漫 / 国创动漫以及其原作或者衍生作品。

# Awiki 如何运行？

Awiki 目前因为经费和技术力不足的原因，只能使用静态博客框架形式运行。

# Awiki 的创建目的是什么？

原是为了自己研究 hexo 主题制作，并分享自己喜欢的番剧。后来因为某件事情导致某部番剧的下架，又导致某事件的发生，决定以投稿的方式召集更多文章：

  • 本站初衷是为了推荐番剧以及标注雷点。
  • 本站欢迎小伙伴共同讨论。
  • 本站欢迎路人著名 / 匿名投稿。

# Awiki 与其他 ACG 相关的 wiki 站点的区别是什么？

Awiki 并不打算只成为单一的词条收集库，更想收集一些中立客观的观后评论，以及追番建议。

# Awiki 目前维护人员

静态框架并不需要多少人力财力，目前仅站长一人维护。

# 收入

Awiki 没有任何收入。

# 缺乏编辑

  1. 由于 wiki 并不在大陆流行，而且 Awiki 依靠 GitHub 托管并不会被百度收录，不容易被人看到。
  2. 大多数中文用户倾向于被动接受内容。
  3. 目前 ACG 圈内新手用户增多，不情愿研究需要稍加学习的 wiki 编辑技巧。

# 技术力缺乏

  1. 目前仅一人维护，刚接触且工作量大、静态网站等原因限制了大多数高级功能。
  2. 网站仅一人维护，可能对于应对危机有很大的漏洞。

# 文案冲突 / 版权冲突

由于 Awiki 仅为刚创建的小站，在文章内容上可能与其他网站发生冲突。而且无法大量创作高质量原创内容，会借助 萌娘百科 等 wiki 网站的资料。

  1. Awiki 承诺尽量保证大多数为原创内容。
  2. Awiki 承诺参考内容均会标记原出处，遵守原出处的版权声明以及其规则。

# 版权信息

Awiki 的全部内容禁止商业使用。
