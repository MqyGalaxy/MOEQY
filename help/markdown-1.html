help:撰写文章

2021-02-26

由于 Awiki 是一个静态框架驱动的网站，需要进行本地编写后上传。请准备好本地编写的工具。

# 文章编写

# 创建文章词条

Awiki 作为番剧推荐网站，所创建的词条应是番剧主题介绍以及推荐，和相关原作或者衍生作品。

  • 正确例：《强袭魔女》应写入番剧主要信息、世界观架构、小说 / 漫画等衍生作品，以及评价推荐。
  • 错误例：宫藤芳佳等角色介绍。
    Awiki 主要作为番剧推荐使用，请不要把页面大多数文笔用在角色介绍上隔壁的萌娘已经干过这活啦。
    可以在番剧介绍的词条中适当地引入角色介绍。

此外，创建词条时请准备好一定的资料，尽可能的撰写原创内容。以免出现低创词条。

  • 比如：该番剧的制作信息 / 世界观架构 / 社区评价 / 注意事项等。
  • 如果仅介绍了番剧或者仅有评价的文章是不符合本站的创作规则的。

# 词条开篇

Awiki 作为一个推荐 wiki 站点，首先要说明自己的文章介绍了什么。
每一篇文章必须有这样的开篇，要让读者第一眼就能了解到文章介绍的番剧说明，

  • 比如：番剧的译名 / 原名 / 副标题（别称）、企划、制作公司等。

# 保持中立观点与客观视角

Awiki 所有的文章均应该体现中立观点以及客观角度地思考和推荐。

  • 编辑者应使用第三人称编写条目。（除非你是原创作者来推荐自己的作品，可使用第一人称）
  • 原则上，编辑者不应该在所编辑的词条中做出个人评价和褒贬，而应以第三人称视角，搜集相关资料和社会观点，使用引述的方式展现给读者对于作品以及其相关内容的评价。

正确例：人们都认为伊藤诚是人渣。
比如 “我最喜欢 XX”“XX 最可爱” 等个人言论不应该出现在文章中。

# 幽默内容使用

Awiki 在推荐番剧时，允许使用一定程度上的幽默性内容主要学习萌娘，写文章开开玩笑还是挺开心的～。
但是，允许编辑此类内容不代表词条编辑可以无节制的添加此类内容。

  • 幽默内容不能违反撰写要求。
    虽然允许使用但不包括黑幕、模糊、删除线等正式内容之外，使用轻松的语句描述番剧的世界观、角色等，或者是带有非严肃性且恰当的玩梗性质。但这些内容同样遵守中立性与客观性的要求，不允许出现违反撰写要求的内容（包括但不限于伪造不属实内容、过分玩笑、刻意编辑不良内容等）。

  • 幽默内容不影响正常阅读。
    幽默内容仅为了词条的趣味性和特色性而存在的表达方式，其存在不能妨碍读者对其他正常内容的阅读：
    其一，保证词条在去除幽默内容后仍然可以正常阅读。要求在编辑时处理幽默内容时，保持其语句的连贯性。例如：“涅普缇努作为紫灵心女神守护着海星国但是平常的工作总是喜欢推给别人，自己却到处摸鱼。”
    其二，不要滥用幽默模板。幽默模板仅限使用于小段的特定内容，包括但不限于剧透、黑历史、不便明言的内容、带有吐槽性质的公认内容。过多的幽默模板反而会导致严重影响读者的阅读体验，应考虑清除不必要的幽默模板。

  • 不要同时使用多个幽默模板
    比如： !!黑幕!!模糊!!!! 等，这样不但会导致读者的阅读体验，亦有可能造成潜在的显示 bug。因此，编辑时不应该套用多个幽默模板。

  • ** 不要 “弹幕式” 编辑幽默模板
    在编辑幽默内容时，一部分编辑者看到前者的吐槽，又想进行回复评论，于是在该内容后添加吐槽的黑幕用于评论。首先该编辑内容通常主观性质强烈，而且影响读者的正常阅读，编辑者不应该在编辑时做出这种行为。如果情势导致模板内的内容出现了变化（比如时间推移导致内容不再适用），应当删除该内容并重写。如果想要进行讨论应当在评论区发表言论。

# 词条主要内容

Awiki 主要作为番剧推荐使用，主题内容请围绕番剧本体展开。如有特殊情况可以参照原作、衍生作品等展开（比如动画化毁原作、二次创作范围等）。

# 外部资料于链接

Awiki 无法避免参考外部资料。如若在撰写文章时借鉴了外部资料（比如萌娘百科等），请在文章的最后注明参考地址。

  • 比如：“Awiki：参考外部资源于链接。”
    参考例：参考链接

# 搜索参考资料

在网络上不仅仅汇集着参考资料，还汇集了各种意见。综合各种意见和评论，基本可以比较全面地看到一个事件以及读者和用户对事件的态度。
在搜索参考资料时，首先将低质量资料淘汰，但也请注意有用的信息点；其次再筛选剩余的文章，提取出精华和信息。
参考资料不代表可以直接复制，应该在参考资料的基础上加以改进，展现出最重要的信息。
最后注意记得注明参考地址。

参考资料来源方式：

  • 观察分析番剧 / 原作本体情节、剧情、世界观，并且在分类中找到适合的分类词条与位置。
  • 网络报道，包括但不限于：搜索引擎、知乎类型网站、视频类型网站、书籍、官网、Wiki 站点等。

词条所需资料：
拥有这些内容后，词条才算是完整的。

  • 音乐：基本介绍、歌曲名、演唱者、作曲 / 编曲、作词 / 填词、歌词、专辑。
  • 动画：基本的动画信息、剧情简介、人物简介、设定简介、制作人员、分集剧情、相关音乐、衍生作品。
  • 小说：基本介绍、剧情简介、人物简介、设定简介、衍生作品。
  • 人物：基本介绍、作品、梗、活动、图片。
  • 用语 / 成句 / 梗：基本介绍、原文、出处、用法、例句。
  • 术语：基本介绍、相关作品。
